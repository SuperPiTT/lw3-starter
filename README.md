## Start project

composer install
npm install
npm run build
npm run dev
php artisan migrate
php artisan serve


### Dependencies

- **[Laravel](https://laravel.com/)**
- **[Jetstream](https://jetstream.laravel.com/)**
- **[Livewire](https://livewire.laravel.com/)**
- **[Tailwind](https://tailwindcss.com/)**
- **[Flowbite](https://flowbite.com/)**
